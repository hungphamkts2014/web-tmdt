# Web TMĐT - Premium Personal Landing Page

Landing page chuyên nghiệp cho dịch vụ tư vấn Phòng Kinh Doanh & Thương mại điện tử.

## Tính năng

- 🎨 Dark mode với thiết kế Glassmorphism cao cấp
- ⚡ Responsive trên mọi thiết bị
- 🎯 7 dịch vụ tư vấn TMĐT
- 📊 Portfolio showcase
- 📝 Form liên hệ tư vấn

## Tech Stack

- HTML5
- TailwindCSS (CDN)
- Google Fonts (Manrope + Inter)
- Material Symbols Icons

## Cách sử dụng

Mở file `index.html` trực tiếp trong trình duyệt hoặc deploy lên hosting tĩnh.
